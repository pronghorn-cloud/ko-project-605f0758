Update to the latest version

 Keep your application up to date with the latest features, improvements, and bug fixes by updating to the newest version of the design system. Breaking changes in v6.0 This major version includes breaking changes. Please review the migration guide carefully before upgrading. View migration guide

Major differences between current and new version

 Version 6.0 introduces several significant improvements and changes:
  • Enhanced accessibility: All components now meet WCAG 2.1 Level AA standards
  • Improved performance: Reduced bundle size by 30% through optimized component code
  • New design tokens: Updated color palette and spacing system for better consistency
  • Modernized components: Refreshed visual design aligned with current web standards
  • Better TypeScript support: Full type definitions for all components and props

Update process

 Follow these steps to update your application:

1. Update package dependencies

 Update the design system package to the latest version:
npm install @abgov/web-components@latest

2. Review breaking changes

 Check the migration guide for a complete list of breaking changes and how to address them in your code.

3. Update component usage

 Update your components to use the new API. Some component props have been renamed or removed.

4. Test your application

 Thoroughly test your application to ensure all components work as expected with the new version.

Need help?
If you encounter any issues during the update process, our team is here to help.
  • Submit an issue on GitHub
  • Join us on Slack: #design-system-support
  • Contact the design system team
